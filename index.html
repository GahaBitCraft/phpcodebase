1. Introduction to PHP

PHP (Hypertext Preprocessor) is a popular open-source scripting language...

  • What is PHP?

    PHP is a server-side scripting language designed for web development. It is embedded in HTML and is used to create dynamic web pages. PHP is widely used due to its simplicity, flexibility, and integration with databases.

  • History of PHP

    PHP was created in 1994 by Rasmus Lerdorf. Initially, it was a set of tools to track visits to his online resume. Over time, it evolved into a full-fledged programming language, with PHP 3.0 marking its official release in 1998.

  • Why use PHP?

    PHP is easy to learn, open-source, and cross-platform. It integrates seamlessly with databases like MySQL, is highly scalable, and has a large community for support. It is ideal for building dynamic and interactive web applications.

  • PHP vs Other Server-Side Languages

    Compared to Python, PHP is more focused on web development. Unlike Java, PHP is easier to learn and deploy. PHP is synchronous, while Node.js is asynchronous, making PHP better for traditional web apps and Node.js better for real-time apps.

2. Getting Started with PHP

To get started with PHP, you need to install PHP, set up a local server, and write your first PHP script...

  • Installing PHP

    Download PHP from the official website or install a local server environment like XAMPP, MAMP, or LAMP, which includes PHP, Apache, and MySQL. Verify the installation by running php -v in the terminal.

  • Setting Up a Local Development Environment (XAMPP, MAMP, or LAMP)

    Install a local server stack like XAMPP (cross-platform), MAMP (macOS), or LAMP (Linux). Place your PHP files in the server's root directory (e.g., htdocs for XAMPP) and access them via http://localhost.

  • First PHP Script

    Create a file named index.php with the following content:

    
<?php
    echo "Hello, World!";
?>

    Save it in the server's root directory and access it via http://localhost/index.php.

3. PHP Syntax and Basics

PHP syntax is simple and flexible. Here are some examples:


<?php
    echo "Hello, World!";
?>
  • Variables and Data Types

    Variables in PHP start with a $ symbol, e.g., $name = "John";. PHP supports data types like strings, integers, floats, booleans, arrays, and objects.

    
<?php
// String
$name = "John";
echo "Name: " . $name;

// Integer
$age = 25;
echo "\nAge: " . $age;

// Float
$price = 19.99;
echo "\nPrice: $" . $price;

// Boolean
$isAvailable = true;
echo "\nAvailable: " . ($isAvailable ? "Yes" : "No");

// Array
$fruits = ["Apple", "Banana", "Cherry"];
echo "\nFavorite Fruit: " . $fruits[0];

// Object
class Car {
    public $make;
    public $model;

    public function __construct($make, $model) {
        $this->make = $make;
        $this->model = $model;
    }
}

$car = new Car("Toyota", "Corolla");
echo "\nCar: " . $car->make . " " . $car->model;
?>
  • Operators in PHP

    PHP supports arithmetic operators (+, -, *, /), comparison operators (==, !=, >, <), and logical operators (&&, ||, !).

    
<?php
// Arithmetic Operators
$a = 10;
$b = 5;
echo "Addition: " . ($a + $b);
echo "\nSubtraction: " . ($a - $b);
echo "\nMultiplication: " . ($a * $b);
echo "\nDivision: " . ($a / $b);

// Comparison Operators
echo "\nEqual: " . ($a == $b ? "True" : "False");
echo "\nNot Equal: " . ($a != $b ? "True" : "False");
echo "\nGreater Than: " . ($a > $b ? "True" : "False");

// Logical Operators
$isTrue = true;
$isFalse = false;
echo "\nLogical AND: " . ($isTrue && $isFalse ? "True" : "False");
echo "\nLogical OR: " . ($isTrue || $isFalse ? "True" : "False");
echo "\nLogical NOT: " . (!$isTrue ? "True" : "False");
?>
  • Control Structures

    PHP provides control structures like if-else, switch, and loops (for, while, foreach) to control the flow of execution.

    
<?php
// If-Else
$age = 20;
if ($age >= 18) {
    echo "You are an adult.";
} else {
    echo "You are a minor.";
}

// Switch
$day = "Monday";
switch ($day) {
    case "Monday":
        echo "\nStart of the week!";
        break;
    case "Friday":
        echo "\nEnd of the work week!";
        break;
    default:
        echo "\nIt's just another day.";
}

// For Loop
for ($i = 1; $i <= 5; $i++) {
    echo "\nNumber: " . $i;
}

// While Loop
$count = 1;
while ($count <= 3) {
    echo "\nCount: " . $count;
    $count++;
}

// Foreach Loop
$colors = ["Red", "Green", "Blue"];
foreach ($colors as $color) {
    echo "\nColor: " . $color;
}
?>
